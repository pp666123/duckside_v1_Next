# duckside
2022年資展國際前端班專題
2023年使用Next及Redux重構
簡介：以鴨子為主題串接投資記帳、模擬投資小遊戲和裝扮遊戲的專題

# duckside功能
1. 會員登入 (第三方、機器人驗證)
2. 會員資料修改
3. 迎賓動畫&美術設計
4. 總覽&數字視覺化
5. 我的計畫
6. 交易紀錄
7. 資產明細
8. 投資成果
9. 模擬投資
10. 關於本站
11. 合作夥伴

# 啟動專案
```bash
yarn start
yarn dev
```
